# 报错

Module not found: Error: Can't resolve 'axios' in 'C:\Users\25589\Desktop\vue-project-01\src'

原因：初次使用axios没有安装相关的插件

解决方法：运行指令 `npm install --save axios`(使用管理员权限)

再次运行 `npm run serve`
